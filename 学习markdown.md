# Markdown语法
## Markdown基本语法
直接写文字。  
敲两个空格再按换行可以显示换行。

敲两次换行键也可以换行，但有点像换了两行 


**在一个句子两侧(前端和后端)加两个星号可以加粗字体**  
__在一个句子两侧(前端和后端)加两个英文下划线也可以加粗字体__  
同理也可以单独对一个**词语**加粗  


*在一个句子两侧(前端和后端)加一个星号可以斜体字体*  
_在一个句子两侧(前端和后端)加一个英文下划线也可以斜体字体_

## 代码块

在代码区域的上一行和最后一行加上三个```可以生成代码块，且其后可以声明代码语言类型  
```C
int main()
{
    while(1)
    {
        printf("Hello World")；
    }
}
```

而对于单独某一个代码中的函数，也有对应的小代码块（小小的也很可爱（bushi））  
就比如`malloc()`这个函数，只需要再希望生成的代码区间两边各加一个 “  ` ”即可
